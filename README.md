# 📊 Data Science & Machine Learning Portfolio

Welcome to my machine learning repository! This project showcases an end-to-end data science portfolio featuring exploratory data analysis (EDA), robust feature engineering, handling class imbalances, regression modeling, and classification systems. 

Across these tasks, pipelines are constructed using `scikit-learn`, `pandas`, `numpy`, and visualized using `matplotlib` and `seaborn`.
